- for loop 안에서 pet name print 하고싶은데 어떻게 하면되나요 T.T

  ```python
  sparky = {
      'kind': 'dog',
      'owner': 'oh',
  }
  draggy = {
      'kind': 'dragon',
      'owner': 'harry',
  }
  thunder = {
      'kind': 'lightning',
      'owner': 'thor',
  }
  
  pets = [sparky, draggy, thunder]
  
  for pet in pets:
      print(pet)  
```
